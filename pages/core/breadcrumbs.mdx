# Breadcrumbs

Simple breadcrumbs component

## Example

<Breadcrumbs>
  <BreadcrumbsItem>Foo</BreadcrumbsItem>
  <BreadcrumbsItem>Bar</BreadcrumbsItem>
  <BreadcrumbsItem>Baz</BreadcrumbsItem>
  <BreadcrumbsItem isCurrentPage>Current page</BreadcrumbsItem>
</Breadcrumbs>
